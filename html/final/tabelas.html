Opções da tag Track

Atributo Valor Descrição
default default Especifica que a faixa deve ser ativada se as preferências do usuário não indicarem que outra faixa seria mais apropriada
kind captions
chapters
descriptions
metadata
subtitles 		Especifica o tipo de faixa de texto
label text Especifica o título da faixa de texto
src URL Obrigatório. Especifica o URL do arquivo de trilha
srclang language_code Especifica o idioma dos dados de texto da faixa (obrigatório se kind = "subtitles")
